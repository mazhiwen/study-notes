# css3

```
新增各种CSS选择器 （:not(.input)：所有class不是“input”的节点）
圆角  （border-radius:8px）
多列布局 （multi-column layout）
阴影和反射 （Shadow\Reflect）
文字特效  （text-shadow）
文字渲染  （Text-decoration）
线性渐变  （gradient）
旋转   （transform）
缩放，定位，倾斜，动画，多背景
例如：transform:\scale(0.85,0.90)\translate(0px,-30px)\skew(-9deg,0deg)\Animation:
```
